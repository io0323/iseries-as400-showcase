■ プロジェクト概要

インテリア専門商社向け基幹システムのRPG IVポートフォリオです。

受発注・物流在庫・会計の3モジュールをFree-format RPG（**FREE）で実装しています。


■ 実装ファイル構成

モジュール ファイル 種別 説明
受発注 ORDPF.DDS 物理ファイル 受注データ（主キー：受注番号）
受発注 ORDLF.DDS 論理ファイル 得意先コード・日付順アクセスパス
受発注 ORDDSPF.DDS ディスプレイファイル 5250端末画面定義（ORDMNU/ORDDET）
受発注 ORDMGR.RPGLE RPG IV CRUD・在庫引当・受注残照会
在庫 ITMPF.DDS 物理ファイル 商品マスタ
在庫 INVPF.DDS 物理ファイル 在庫（商品+ロケーション複合キー）
在庫 INVLF.DDS 論理ファイル ロケーション順アクセスパス
在庫 INVMGR.RPGLE RPG IV 入出庫・在庫調整・AllocateStock
会計 ARPF.DDS 物理ファイル 売掛金（消費税・入金消込）
会計 APPF.DDS 物理ファイル 買掛金（仕入・支払処理）
会計 ACCMGR.RPGLE RPG IV 入金消込・支払処理・残高照会
共通 UTILSRV.RPGLE SRVPGM 日付変換・採番・税額計算ユーティリティ
共通 ERRSRV.RPGLE SRVPGM エラー管理・バリデーション共通処理

■ RPG IV 実装の特徴

**FREE
// Free-format RPG IV（**FREE宣言）
Ctl-Opt DftActGrp(*No) ActGrp('ORDMGR') BndDir('QC2LE');

// ファイル定義（DCL-F）
DCL-F ORDPF    Keyed Usage(*Update : *Delete : *Output : *Input);
DCL-F ORDDSPF  WorkStn;

// データ構造（DCL-DS）
DCL-DS OrdDs LikeRec(ORDREC : *All);

// プロシージャ（DCL-PROC）
DCL-PROC GenOrderNo;
  DCL-PI *N Char(10);
  END-PI;
  // 最終受注番号+1で採番
  Return 'ORD' + %EditC(LclSeq : 'Z');
END-PROC;
